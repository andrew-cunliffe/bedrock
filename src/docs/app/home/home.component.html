Bedrock

by Venzra

The design components we use to build the Venzra commerce platform

About

Bedrock is the codename for our Angular Component library. Venzra has been built over the past 2 years and we have extracted these from our commerce platform and we have been making several improvements while doing this, which we intend to bring back across to the platform.

Usage of these components is made available under the MIT license.

Every component has been tested to work on all current browsers. Support for older browser is being improved but potentially some issues may exist.

Getting Started

To install simply add bedrock as a project dependency.

            npm i @venzra/bedrock

The components are built to be used on an as needed basis, benefiting from Angular's tree shacking capability. Once installed use our documentation to enable as required and use the components to build your website.
