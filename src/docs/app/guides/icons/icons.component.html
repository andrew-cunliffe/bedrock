Icons

The full set of icons available can be found on the FontAwesome website.

Import Module

While FontAwesome is a dependency for Bedrock, to use the full range of icons you need to import the library into your own project module to use them.

Each icon you wish to use must be imported, the format to import is camelCase so arrow-up would be imported as faArrowUp this allows for the best efficiency as only those icons used will be compiled into your project, further information can be found on the FortAwesome project.

Note: We only have a dependency on the solid icons for the regular, branding or pro icons you will need to add these dependencies accordingly.
