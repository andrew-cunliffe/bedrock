Icons

The full set of icons available can be found on the FontAwesome website.

Import Module

While FontAwesome is a dependency for Bedrock, to use the full range of icons you need to import the library into your own project module to use them.

Each icon you wish to use must be imported, the format to import is camelCase so arrow-up would be imported as faArrowUp this allows for the best efficiency as only those icons used will be compiled into your project, further information can be found on the FortAwesome project.

Note: We only have a dependency on the solid icons for the regular, branding or pro icons you will need to add these dependencies accordingly.

        import {{ '{' }} library {{ '}' }} from '@fortawesome/fontawesome-svg-core';
        import {{ '{' }}
            faArrowUp,
            faArrowDown,
        {{ '}' }} from '@fortawesome/free-solid-svg-icons';
        
        @NgModule({{ '{' }}
            imports: [
                FontAwesomeModule,
            ],
        {{ '}' }})
        export class ModuleName {{ '{' }}
            constructor() {{ '{' }}
                library.add(
                    faArrowUp,
                    faArrowDown,
                );
            {{ '}' }}
        {{ '}' }}

Basic usage (arrows)

        <fa-icon icon="arrow-up"></fa-icon>
        <fa-icon icon="arrow-left"></fa-icon>
        <fa-icon icon="arrow-right"></fa-icon>
        <fa-icon icon="arrow-down"></fa-icon>

Icon sizes

        <fa-icon icon="square" size="1x"></fa-icon>
        <fa-icon icon="square" size="2x"></fa-icon>
        <fa-icon icon="square" size="3x""></fa-icon>
        <fa-icon icon="square" size="4x"></fa-icon>
        <fa-icon icon="square" size="5x"></fa-icon>

Advanced options

        <fa-icon icon="spinner" [spin]="true" size="2x"></fa-icon>
        <fa-icon [icon]="['fab', 'angular']" rotate="180" size="2x"></fa-icon>
        
        <fa-layers size="2x">
            <fa-icon icon="square"></fa-icon>
            <fa-icon [icon]="['fab', 'angular']" [inverse]="true" transform="shrink-4"></fa-icon>
        </fa-layers>
        
        <fa-layers [fixedWidth]="true" size="2x">
            <fa-icon icon="envelope"></fa-icon>
            <fa-layers-counter content="99+"></fa-layers-counter>
        </fa-layers>
