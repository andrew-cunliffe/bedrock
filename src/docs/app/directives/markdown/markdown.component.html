Markdown

Often holding text content as markdown enables more flexibility and improved searching, this directive enables markdown to be formatted as HTML.

Include 3rd party library

To use the markdown module an additional library must be installed and included within the scripts section of your angular.json.

        npm i marked
    
        "scripts": [
            "./node_modules/marked/lib/marked.js"
        ]

Import Module

To use the components here you first need to include this module in your application @NgModule

        import {{ '{' }} RockMarkdownModule {{ '}' }} from '@venzra/bedrock';
        
        @NgModule({{ '{' }}
            imports: [
                RockMarkdownModule
            ]
        {{ '}' }})
        export class ModuleName {{ '{ }' }}
