Expansion

Use expansion panels to show and hide detailed content or optional forms.

Import Module

To use the components here you first need to include this module in your application @NgModule

        import {{ '{' }} RockExpansionModule {{ '}' }} from '@venzra/bedrock';
        
        @NgModule({{ '{' }}
            imports: [
                RockExpansionModule
            ]
        {{ '}' }})
        export class ModuleName {{ '{ }' }}

        </rock-expansion>
