Input

All forms require users to input data, these are the building blocks for basic inputs and text fields.

Import Module

To use the components here you first need to include this module in your application @NgModule

        import {{ '{' }} RockInputModule {{ '}' }} from '@venzra/bedrock';
        
        @NgModule({{ '{' }}
            imports: [
                RockInputModule
            ]
        {{ '}' }})
        export class ModuleName {{ '{ }' }}

Simple form fields

        <rock-input label="First name">
            <input rockInput id="firstName" type="text" formControlName="firstName">
        </rock-input>
        
        <rock-input label="Last name">
            <input rockInput id="lastName" type="text" formControlName="lastName">
        </rock-input>

Form fields with error messages

        <rock-input label="Email address">
            <input rockInput id="email" type="text" formControlName="email">
            <rock-input-error></rock-input-error>
        </rock-input>
        
        <rock-input label="Password">
            <input rockInput id="password" type="password" formControlName="password">
            <rock-input-error></rock-input-error>
        </rock-input>

Textarea input example
