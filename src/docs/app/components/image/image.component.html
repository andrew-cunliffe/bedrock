Image

Input to support image fields on a form, can automatically upload or submit with your form. This input uses HTML canvas to resize and reformat images, by default images will have a maximum of 250x250px (ratio is maintained) and converted to JPEG. This options can then be over-ridden, see the examples below.

Include 3rd party library

This input field requires the inclusion of the EXIF library within your project and included within the scripts section of your angular.json.

Import Module

To use the components here you first need to include this module in your application @NgModule

Basic image input

Any files that are dragged to the input will be available as blob data on the form. Also included in the form field is the file name, size and mime type (after any conversion).

Extended attributes

Populated image input

When setting an existing value for an image, use a reference to an existing file viewable on the internet. This can be a fully qualified path or a relative path. Detection of a change and managing a change should be managed during form submission.
