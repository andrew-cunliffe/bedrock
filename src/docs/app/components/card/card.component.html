Card

Add cards to give higher impact content to your site.

Import Module

To use the components here you first need to include this module in your application @NgModule

        import {{ '{' }} RockCardModule {{ '}' }} from '@venzra/bedrock';
        
        @NgModule({{ '{' }}
            imports: [
                RockCardModule
            ]
        {{ '}' }})
        export class ModuleName {{ '{ }' }}
