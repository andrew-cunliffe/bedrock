Button

Import Module

To use the components here you first need to include this module in your application @NgModule

        import {{ '{' }} RockButtonModule {{ '}' }} from '@venzra/bedrock';
        
        @NgModule({{ '{' }}
            imports: [
                RockButtonModule
            ]
        {{ '}' }})
        export class ModuleName {{ '{ }' }}

Examples

Use the following examples to add buttons to your site, for forms / links / actions.

Standard Buttons

        <button type="button" rock-button>Primary</button>
        <button type="button" rock-button class="accent">Accent</button>
        <button type="button" rock-button class="warn">Warning</button>

Disabled Buttons

        <button type="button" rock-button [disabled]="true">Primary</button>
        <button type="button" rock-button class="accent" [disabled]="true">Accent</button>
        <button type="button" rock-button class="warn" [disabled]="true">Warning</button>

Raised Buttons

        <button type="button" rock-raised-button>Primary</button>
        <button type="button" rock-raised-button class="accent">Accent</button>
        <button type="button" rock-raised-button class="warn">Warning</button>

Stroked Buttons

        <button type="button" rock-stroked-button>Primary</button>
        <button type="button" rock-stroked-button class="accent">Accent</button>
        <button type="button" rock-stroked-button class="warn">Warning</button>
