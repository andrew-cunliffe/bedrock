Alert

Alerts can be used to provide warnings or additional information to users.

Import Module

To use the components here you first need to include this module in your application @NgModule

        import {{ '{' }} RockAlertModule {{ '}' }} from '@venzra/bedrock';
        
        @NgModule({{ '{' }}
            imports: [
                RockAlertModule
            ]
        {{ '}' }})
        export class ModuleName {{ '{ }' }}
